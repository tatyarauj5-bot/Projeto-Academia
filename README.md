# Projeto de Banco de Dados: Sistema de Gestão de Academia

Este repositório contém o Modelo Físico (SQL) para o sistema de gestão de academia, desenvolvido como parte da Atividade 4 da disciplina de Banco de Dados.

## Estrutura do Projeto

O código SQL está modularizado em arquivos, seguindo as melhores práticas de organização:

1.  **1_DDL_Criacao_Tabelas.sql**: Contém a Lógica de Definição de Dados (CREATE TABLE com PK e FK).
2.  **2_DML_Insercao_Dados.sql**: Contém a Lógica de Manipulação de Dados (INSERT INTO com dados de exemplo).
3.  **3_DQL_Consultas.sql**: Demonstração de Consultas (SELECT) para relatórios.
4.  **4_DML_Atualizacao.sql**: Demonstração de comandos de UPDATE e DELETE.

## Instruções de Execução

O código deve ser executado em um ambiente de banco de dados relacional (ex: MySQL ou PostgreSQL) na seguinte ordem sequencial: 1_DDL seguido por 2_DML.
